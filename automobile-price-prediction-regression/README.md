Automobile Price Prediction using Machine Learning

# Overview

This project demonstrates a complete supervised machine learning workflow using the Automobile Price dataset. It covers the entire process from data preparation and model building to evaluation and hyperparameter tuning using Scikit-learn.
The primary objective is to predict automobile prices using various regression techniques while understanding concepts such as overfitting, underfitting, polynomial feature engineering, regularization, cross-validation, and model selection.

# Objectives

* Prepare and clean numerical automobile data
* Build Linear Regression models
* Perform Multiple Linear Regression
* Apply Polynomial Feature Engineering
* Detect overfitting and underfitting
* Evaluate models using train/test split
* Perform K-Fold Cross Validation
* Generate cross-validated predictions
* Implement Ridge Regression
* Tune hyperparameters using GridSearchCV
* Compare model performance using the R² score
